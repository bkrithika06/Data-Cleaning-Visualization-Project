##### **Data Cleaning \& Visualization Project**



###### **Objective**



This project focuses on cleaning and visualizing raw Netflix dataset using Python.



\-> Libraries Used



\* Pandas

\* Matplotlib

\* Seaborn



###### **Features**



\* Removed duplicates

\* Handled missing values

\* Processed data

\* Visualized insights



###### **Visualizations**



\* Movies vs TV Shows

\* Top Countries

\* Ratings Distribution

\* Content Added Over Years



###### **Outcome**



Learned data cleaning, preprocessing, visualization, and storytelling with data.
